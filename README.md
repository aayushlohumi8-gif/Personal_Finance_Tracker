# Personal Finance Tracker

A web-based personal finance tracker built using Flask, HTML, CSS, and SQLite.

## Features
- User Login & Signup
- Add Income
- Add Expenses
- Expense Tracking
- Dashboard with Charts

## Technologies Used
- Python (Flask)
- HTML/CSS
- SQLite
- Chart.js
